## 2. 노션 운영 시스템 (이미 완성됨 — 건드릴 땐 이 규칙 준수)

- **"💳 결제·정산 내역" DB**: 모든 결제 통합 관리. 카드/현금/계좌이체 필터 뷰 있음.
  - 결제 1건 = 4주 1사이클 = 회분 4. 가격: 1과목 50만 원, 두 과목 80만 원.
  - 과외와 불꽃캠프는 결제의 **'구분' 필드**로 분리. 과외→불캠 전환율 대시보드 있음.
- **"불꽃캠프 원비" DB**: 학생별 수강권 잔액 관리.
  - 잔여회수 = 충전회수(결제 회분 합) − 누적회(출석 수).
  - 충전(결제 입력)은 사용자가 직접, 차감(출석 → '누적회' 갱신)은 Claude가 자동으로 수행.
- **상담 신청**: 홈페이지 → 노션 폼(과외 리드 트래커 DB의 폼 뷰)이 **직접 기록**.
  Make/Formspree 등 외부 자동연동은 전부 폐기됨 — 다시 제안하지 말 것.
- **워크스페이스 구조** (2026-07-02 재정리): 최상위 "🔥 불꽃캠프-과외" 아래 5개가 전부.
  1. 🎛️ 대시보드 — 모든 실무 입력·조회. **비즈니스 플로우 순서의 linked view 스택** (2026-07-02 재구성):
     ① 상담 퍼널(리드 트래커 보드 — 상태: 신규/상담중/상담만 그침/과외 진행·중단/불캠 진행·중단/보류)
     ② 과외 리스트(**표 하나, 그룹핑 없음** — '🔥 불캠 전환'은 체크박스 열로만) ③ 원비(잔여회수) ④ 결제·정산 ⑤ 시험·시험지
     ⑥ 점수(**응시일 그룹 = 드라이브 시트처럼 날짜별**) ⑦ 카톡방 운영 ⑧ 조교 HR ⑨ 불캠 학생 명단
     — 대시보드에 설명 텍스트 넣지 말 것 (헤더 2줄이 전부)
  2. 📸 인스타 운영 (조교) — 가이드 + 콘텐츠 보드 DB
  3. 📚 자료실 — 읽는 문서 전부. **새 문서는 무조건 여기에 생성**
  4. 🗃️ DB 원본 — 모든 DB 원본 (결제·정산, 원비, 학생, 출석부, 리드 트래커, 점수 마스터, 시험,
     💬 카톡방 운영(공지/정오표/주간 자료), 🧑‍🏫 조교 HR(채용·급여) 등)
  5. 🗄️ 아카이브 — 옛 메모·폐기 시스템. 참고 금지, 삭제도 금지
  - 새 최상위 페이지·새 카테고리를 만들지 말 것. 이 5개 구조 유지.

## 2-0. 시험·점수 시스템 (2026-07-02 구축)

- **원칙**: 시험지 파일 = 드라이브(링크만 노션에), 점수 데이터 = 노션에만. 드라이브 점수 시트("2026 불꽃사탐" 등)는 과거 기록 보관용 동결 — 점수 이중 입력 금지.
- **이관 완료 (2026-07-02)**: "2027 학생 시트"의 4/18~6/28 점수 518건 + 시험 64개를 노션으로 이관 (메모에 '2027 학생 시트에서 이관' 표기). 원비 미등록이라 학생 연결 없이 들어간 이름: 신우진·유진모·한준석·김지후·이수민·왕서준·김서진·준우·홍예준·윤원준·정서윤·신규림 (정서윤↔장서윤, 신규림↔김규림은 오타 의심 — 사용자 확인 필요). "2026 불꽃사탐" 시트는 작년(2026학년도) 기록이라 이관하지 않음.
- **"🧾 시험" DB**: 시험 1개 = 1행 (시험명·과목·유형(OX/기출/사설)·시험일·시험지 드라이브 링크).
  - '기반 기출' 자기참조 관계: OX 모의고사는 이전 주 기출 2개로 제작 (3중 복습 2차).
  - 주간 구성 고정: 사문 = OX 1 + 기출 2 + 사설 2 / 생윤 = OX 2 + 기출 4.
- **"🎯 점수 마스터" DB**: 학생×시험 = 1행. **행 제목(title '학생')은 "이름 · 시험명" 형식**(예: "차현서 · 사문 23-3") — 학생 페이지의 '시험 기록' 목록이 읽히도록. 새 점수 입력 시에도 이 형식 유지. '학생 연결'(→원비 DB, 양방향 '시험 기록') + '시험 연결'(→시험 DB) + 1·2·3회차/최종. 상승폭·만점도달 자동 수식.
  - 원비 DB에서 학생 페이지를 열면 '시험 기록'으로 시험별 재시험 점수가 보임.
  - 대시보드 ⑥ 점수 뷰는 '응시일' 그룹(날짜별), 점수 마스터 DB 자체의 '학생별 추이'·'시험별 통계' 뷰는 관계 그룹.
  - 기존 12행(이유찬·한규연 등)은 현행 학생 명단과 이름 불일치 — 작년/샘플 데이터로 추정, 관계 없이 유지.
- 수업 루프: 모의고사 → 조교 1:1 피드백 → 그날 100점까지 재시험. 3중 복습: 이번 주 기출 → 다음 주 OX 재료 → 4주 후 재응시.

- 불변 규칙: 학생 신상 동의 없이 노출 금지 / 성적 보장 표현 금지 / 타 강사·업체 언급 금지 / 숫자(성적·데이터) 포함 콘텐츠만 장쌤 검수 필수. 나머지는 조교 재량.
